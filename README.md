# ClimbNow

🌐 [https://climbnow.ru](https://climbnow.ru)
🌐 [https://climbnow-skondor.amvera.io/](https://climbnow-skondor.amvera.io/)

ClimbNow - это веб-приложение для отображения соревнований с сайта Федерации Скалолазания России [https://c-f-r.ru/live/2602vrn/index.html](https://c-f-r.ru/live/2602vrn/index.html). Приложение позволяет пользователям одновременно просматривать таблицы результатов по различным дисциплинам, фильтровать и подсвечивать скалолазов своей команды.

## Стэк

NextJS, React, TypeScript, MobX, Tanstack React Query, Axios, TailwindCSS

## Особенности

- Отображение результатов соревнований по дисциплинам (трудность и боулдеринг)
- Подсветка и фильтрация участников выбранной команды
- Адаптивный дизайн для работы на различных устройствах
- Автоматическое обновление данных при изменении параметров

## Использование

1. Введите код соревнований в поле "код соревнований" (например, `2602vrn`)
2. При необходимости, укажите название команды в поле "команда" (например, `СПБ`)
3. Активируйте чекбокс "только команда", чтобы отфильтровать результаты по указанной команде
4. Активируйте чекбокс "только онлайн", чтобы показывать только группы которые в данный момент лезут (на c-f-r.ru данные не всегда отображается корректно)
5. Приложение автоматически обновляет данные, группы - раз в 10 минут, открытые реультаты - раз в 1 минуту

## Установка и запуск

### Шаги установки

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ваш-репозиторий/climb-now.git
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите разработческий сервер:

```bash
npm run dev
```

4. Откройте приложение в браузере по адресу:

```
http://localhost:3000
```

## API

Приложение использует внешнее API [https://c-f-r.ru/live/2602vrn/index.html](https://c-f-r.ru/live/2602vrn/index.html) для получения данных о результатах соревнований.

Основные эндпоинты:

- `/api/groups` - получение списка групп соревнований
- `/api/results` - получение результатов по дисциплинам
